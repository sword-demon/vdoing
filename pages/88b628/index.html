# MP 的自动填充

每次我们修改，都需要调用xxSet方法来进行修改，MP 中有它特有的方式来实现和Set方法相同的效果

我们在表里添加 2 个字段：datetime类型的create_time和update_time

这里我们修改了表结构，我们也需要修改表对应的实体类添加对应的属性

package com.wjstar.demomptest.entity;

import com.baomidou.mybatisplus.annotation.IdType;
import com.baomidou.mybatisplus.annotation.TableId;
import lombok.Data;

import java.util.Date;

@Data
public class User {
    @TableId(type = IdType.ASSIGN_ID)
    private Long id;
    private String name;
    private Integer age;
    private String email;

    private Date createTime;
    private Date updateTime;
}

# 给实体类添加自动填充注解

需要进行自动填充，我们还需要在实体类需要自动填充的属性上添加注解

@Data
public class User {
    @TableId(type = IdType.ASSIGN_ID)
    private Long id;
    private String name;
    private Integer age;
    private String email;

    @TableField(fill = FieldFill.INSERT)
    private Date createTime;

    // 添加和修改的时候都有值
    @TableField(fill = FieldFill.INSERT_UPDATE)
    private Date updateTime;
}

# 实现元对象处理器接口

创建一个类实现接口，实现接口两个方法：

  1. 一个方法添加执行
  2. 一个方法修改执行

最后还得将这个类的对象创建交给 Spring 来控制，即加上@Component注解

package com.wjstar.demomptest.handler;

import com.baomidou.mybatisplus.core.handlers.MetaObjectHandler;
import org.apache.ibatis.reflection.MetaObject;
import org.springframework.stereotype.Component;

import java.util.Date;

@Component
public class MyMetaObjectHandler implements MetaObjectHandler {

    /**
     * MP 执行添加操作的时候这个方法就会执行
     *
     * @param metaObject
     */
    @Override
    public void insertFill(MetaObject metaObject) {
        this.setFieldValByName("createTime", new Date(), metaObject);
    }

    /**
     * MP 执行修改操作的时候这个方法就会执行
     *
     * @param metaObject
     */
    @Override
    public void updateFill(MetaObject metaObject) {
        this.setFieldValByName("updateTime", new Date(), metaObject);
    }
}
