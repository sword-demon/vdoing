线程的常见方法

# 线程的常见方法

  1. start(): 启动当前线程，表面上调用start方法，实际在调用线程里的run方法
  2. run(): 线程类继承Thread类或者实现Runnable接口的时候，都要重新实现run方法，run方法里的是线程要执行的内容
  3. currentThread: Thread类中的一个静态方法，获取当前正在执行的线程
  4. setName: 设置线程的名称
  5. getName: 读取线程的名称

# 设置优先级

  1. 同优先级的线程，采取的策略就是先到先服务，使用时间片策略

  2. 如果优先级别高，那就被 CPU 调度的概率就高

  3. 级别：1-10 默认的级别为 5

Thread类源码级别

public final static int MIN_PRIORITY = 1;

/**
     * The default priority that is assigned to a thread.

     * The maximum priority that a thread can have.
     */

# join 方法

当一个线程调用了join方法，这个线程就会被先执行，它执行结束后才可以去执行其余的线程。

警告

必须先start，再join才有效

join就感觉就像”半路杀出个程咬金“！

package com.wx.virus;

public class TestThread extends Thread {

    public TestThread(String name) {
        super(name);
    }

    @Override
    public void run() {
        for (int i = 0; i < 10; i++) {
            System.out.println(this.getName() + "---" + i);
        }
    }
}

class Test1 {
    public static void main(String[] args) throws InterruptedException {
        for (int i = 1; i <= 100; i++) {
            if (i == 6) {
                // 创建一个子线程
                TestThread tt = new TestThread("子线程");
                tt.start();
                tt.join();
            }
            System.out.println("main---" + i);
        }
    }
}

package com.wx.virus;

import java.text.DateFormat;
import java.text.SimpleDateFormat;
import java.util.Date;

public class Test03 {
    public static void main(String[] args) {
        // 2. 定义一个时间格式
        DateFormat df = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");

        while (true) {
            // 1. 获取当前时间
            Date d = new Date();
//            System.out.println(d);

            String format = df.format(d);
            System.out.println(format);
            // 延迟1秒打印一次
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }
    }
}

# 设置伴随线程

将子线程设置为主线程的伴随线程，主线程停止的时候，子线程也不会继续执行了

案例：古代皇上(主线程)驾崩了 -> 陪葬妃子(伴随线程)

警告

先设置伴随线程，再调用start

package com.wx.virus.t1;

public class TestThread extends Thread {
    @Override
    public void run() {
        for (int i = 1; i <= 20; i++) {
            System.out.println("子线程---" + i);
        }
    }
}

class Test {
    public static void main(String[] args) {
        // 创建并启动子线程
        TestThread tt = new TestThread();
        // 设置伴随线程 先设置再启动
        tt.setDaemon(true);
        tt.start();

        // 主线程中还要输出1-10的数组
        for (int i = 1; i <= 10; i++) {
            System.out.println("main---" + i);
        }
    }
}

# stop 方法

提前结束了线程的运行，进入死亡。

public class Demo {
    public static void main(String[] args) {
        for (int i = 1; i < 100; i++) {
            if (i == 6) {
                // 过期方法：不建议使用
                Thread.currentThread().stop();
            }
            System.out.println(i);
        }
    }
}
