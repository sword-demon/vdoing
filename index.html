# 在 struct 中使用泛型

# 泛型

定义不局限名称，可以写任意的一个字母，最好是大写，因为从编程以来，认识的使用泛型的基本都是使用的大写字母，但是肯定会有一对<>进行包裹。

背景：

现在有一个积分平台，老同学使用的是i32类型作为积分的类型；现在有一个需求要支持f32类型的，这就需要泛型来支持。
