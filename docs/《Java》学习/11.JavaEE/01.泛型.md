## 泛型

> 泛型就相当于标签

形式：`<>`

:::tip

集合容器类在设计阶段/声明阶段不能确定这个容器到底实际存什么类型的对象，所以在 JDK1.5 之前只能把元素类型设计为`Object`，JDK1.5 之后使用泛型来解决。因为这个时候除了元素类型不确定，其他的部分是确定的，因此此时把元素的类型设计成一个参数，这个类型参数叫做**泛型**

`Collection<E>，List<E>，ArrayList<E>`这个`<E>`就是类型参数，即泛型。

:::
